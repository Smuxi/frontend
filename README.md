# KataBooks Project

This project consists of a Java Spring Boot API and a React frontend.

---

## Prerequisites

- Java 17+
- Node.js (v18+) and npm
- Gradle

---

## 1. Start the API

Navigate to the backend directory (if applicable) and run:

```bash
./gradlew bootRun
```

## 2. Start the React Frontend

Navigate to the frontend directory:

```bash
cd frontend
npm install
npm start
